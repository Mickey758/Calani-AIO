# Calani AIO
An all in account checker made in python. Made with multichecking and more. This was made to demonstrate the flaws in certain website API's.

# Progress
- [x] Log Checking Mode
- [x] Cui Checking Mode
- [ ] Auto Updater
- [ ] Config File
- [ ] Save Remaining Lines
- [x] Multichecking
- [ ] In-app Settings
